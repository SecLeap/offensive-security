# 11｜对抗模拟与攻击路径

目标：依据威胁情报和明确规则，在模拟环境中验证检测、响应和关键业务防护能力。

## 主题

- [ ] 对抗模拟范围、目标、成功标准、停止条件和安全护栏。
- [ ] ATT&CK 技术映射、对手画像与可观测行为设计。
- [ ] 初始访问、执行、持久化、权限提升、横向移动等阶段的防守视角理解。
- [ ] 攻击路径：身份、网络、配置、应用与数据之间的依赖关系。
- [ ] Purple Team：检测假设、验证、差距记录、规则优化与回归测试。
- [ ] 行动复盘：时间线、检测覆盖、响应摩擦、风险与改进优先级。

## 达标产物

- [ ] 对抗模拟计划和安全护栏模板。
- [ ] ATT&CK 覆盖矩阵与检测验证模板。
- [ ] 攻击路径图与整改跟踪模板。
