# 16｜参考资料、笔记与复盘

目标：把外部知识转化为可检索、可验证、可更新的长期资产。

## 分类

- [ ] 标准与框架：OWASP、NIST、MITRE ATT&CK、CIS、ISO 等。
- [ ] 漏洞与公告：CVE、CWE、厂商安全公告、补丁说明与披露报告。
- [ ] 技术笔记：协议、平台、漏洞类别、工具、检测和修复卡片。
- [ ] 课程与书籍：学习计划、要点摘录、练习和验证结果。
- [ ] 案例复盘：攻击链、根因、检测缺口、整改成效和通用教训。
- [ ] 术语表：中英文术语、缩写、上下文与相关主题链接。

## 维护规则

- [ ] 每条引用记录来源、发布日期/版本、访问日期与适用范围。
- [ ] 笔记优先记录自己的理解和可验证结论，避免整段转载。
- [ ] 对时效性强的内容标记复核日期，并定期清理失效链接。
