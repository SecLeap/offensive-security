# 12｜检测、响应与防御工程

目标：将攻击面和验证结果转化为可监控、可响应、可持续改进的防御控制。

## 主题

- [ ] 日志体系：端点、身份、网络、应用、云和审计日志的采集与质量。
- [ ] 检测工程：用例、规则、基线、行为分析、误报管理和版本控制。
- [ ] 威胁狩猎：假设驱动、数据源、调查记录和证据关联。
- [ ] 事件响应：分级、遏制、取证、根除、恢复、沟通和复盘。
- [ ] 安全控制：硬化、补丁、WAF、EDR、DLP、访问控制与密钥管理。
- [ ] 指标：覆盖率、检测时延、响应时延、误报率和修复闭环。

## 达标产物

- [ ] 关键资产日志与检测用例矩阵。
- [ ] 事件响应运行手册与演练模板。
- [ ] 控制有效性验证与例外管理流程。
