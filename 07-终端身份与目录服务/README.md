# 07｜终端、身份与目录服务

目标：理解端点安全、身份边界和目录服务中权限关系对攻击路径的影响。

## 主题

- [ ] Windows 安全架构：账户、令牌、服务、注册表、日志与防护机制。
- [ ] Linux 安全架构：用户组、sudo、PAM、systemd、cron、日志与能力集。
- [ ] Active Directory：域、林、OU、GPO、Kerberos、委派、信任与权限继承。
- [ ] 身份与访问管理：SSO、MFA、特权访问、服务账户、密钥与轮换。
- [ ] 端点防护：EDR、应用控制、补丁、硬化、磁盘加密和设备管理。
- [ ] 攻击路径分析：身份、配置、凭据、网络可达性与高价值资产关系。

## 达标产物

- [ ] Windows/Linux 基线审计清单。
- [ ] AD 权限关系和高风险配置检查表。
- [ ] 特权账户治理与检测建议模板。
