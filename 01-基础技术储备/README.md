# 01｜基础技术储备

目标：具备阅读协议、理解应用运行方式、编写小型自动化脚本和维护实验环境的能力。

## 主题

- [ ] 计算机组成：进程、线程、内存、文件系统、权限模型。
- [ ] 网络基础：OSI/TCP-IP、路由、DNS、ARP、TCP/UDP、IPv4/IPv6。
- [ ] Web 基础：HTTP、TLS、Cookie、浏览器同源策略、代理与缓存。
- [ ] 操作系统：Linux 与 Windows 的用户、服务、日志、网络和权限。
- [ ] 编程与脚本：Python、Shell、PowerShell、正则、JSON/YAML、SQL 基础。
- [ ] 数据库与中间件：关系型/非关系型数据库、消息队列、缓存基础。
- [ ] 工程基础：Git、依赖管理、虚拟化、容器、CI/CD 基本概念。
- [ ] 密码学基础：哈希、对称/非对称加密、签名、证书、密钥生命周期。

## 达标产物

- [ ] 一份协议与常见端口速查表。
- [ ] 一份 Linux/Windows 排障与日志定位笔记。
- [ ] 一份安全自动化脚本编码规范。
