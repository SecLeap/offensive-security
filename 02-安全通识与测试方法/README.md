# 02｜安全通识与测试方法

目标：在合法授权和风险可控的前提下，能够规划、执行、记录和复测一次安全测试。

## 主题

- [ ] CIA、最小权限、纵深防御、零信任与安全设计原则。
- [ ] 威胁建模：资产、信任边界、攻击面、威胁主体与缓解措施。
- [ ] 渗透测试流程：范围确认、规则约定、信息收集、验证、报告、复测。
- [ ] 漏洞生命周期：发现、复现、定级、披露、修复、验证与复盘。
- [ ] 风险评级：影响、可能性、可利用性、业务上下文与 CVSS。
- [ ] 证据管理：时间线、截图/日志、复现条件、脱敏与证据完整性。
- [ ] 法律、合规和职业伦理：授权、隐私、数据最小化、停止条件。

## 达标产物

- [ ] 测试授权与规则约定（RoE）模板。
- [ ] 测试计划、发现记录、复测记录模板。
- [ ] 风险评级口径与报告质量检查表。
