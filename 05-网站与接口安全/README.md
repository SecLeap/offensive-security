# 05｜Web 与 API 安全

目标：理解现代 Web 应用、API 和身份流的攻击面，并能将发现转换为修复与检测建议。

## 主题

- [ ] 架构与边界：前后端分离、反向代理、CDN、WAF、微服务与服务间认证。
- [ ] 身份认证：会话、JWT/OAuth/OIDC、MFA、找回流程与账户生命周期。
- [ ] 授权：对象级、功能级、租户隔离、水平/垂直越权与业务权限模型。
- [ ] 输入与输出：注入、文件处理、反序列化、模板、SSRF、跳转与解析差异。
- [ ] 浏览器安全：XSS、CSRF、CORS、CSP、Cookie 属性与前端供应链。
- [ ] 业务逻辑：状态机、金额/库存、并发、风控、邀请与审批流。
- [ ] API：REST、GraphQL、gRPC、Webhook、限流、幂等、版本与错误处理。

## 达标产物

- [ ] Web/API 测试清单（含认证、授权、业务逻辑）。
- [ ] 常见问题的检测信号与修复模式表。
- [ ] API 安全设计评审清单。
