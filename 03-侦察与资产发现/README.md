# 03｜侦察与资产发现

目标：在批准范围内识别资产、暴露服务、技术栈和潜在风险优先级。

## 主题

- [ ] 范围解析：域名、IP、云账号、应用、第三方与排除项。
- [ ] 资产台账：所有者、业务重要性、环境、数据等级和暴露位置。
- [ ] 被动信息收集：公开信息、证书、DNS、代码与泄露暴露面审查。
- [ ] 主动验证：授权扫描、服务识别、版本核验与变更控制。
- [ ] Web/API 攻击面：入口、参数、身份流、上传、管理界面、回调。
- [ ] 优先级：互联网暴露、弱身份边界、高价值数据、已知风险与可达性。

## 达标产物

- [ ] 资产发现清单与范围核验流程。
- [ ] 暴露面优先级模型。
- [ ] 误报、重复资产和第三方资产处理规则。
