# 15｜实验室、工具与可复现性

目标：建立隔离、可回滚、可审计的学习与验证环境。

## 主题

- [ ] 实验室架构：虚拟化、隔离网络、快照、资源配额与备份。
- [ ] 靶场与训练：公开靶场、CTF、故意脆弱应用和自建测试目标。
- [ ] 工具治理：用途、版本、来源、许可证、校验值、更新和替代方案。
- [ ] 自动化：任务编排、输入校验、限速、日志、异常处理和安全开关。
- [ ] 数据管理：测试数据、敏感信息脱敏、样本隔离、保留与销毁。
- [ ] 可复现性：环境说明、前置条件、版本锁定、运行记录与清理步骤。

## 达标产物

- [ ] 实验室部署与回滚指南。
- [ ] 工具登记表和安全使用规范。
- [ ] 实验记录与数据清理检查表。
