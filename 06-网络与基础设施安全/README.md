# 06｜网络与基础设施安全

目标：识别网络服务、边界设备、基础组件和运维面中的配置与身份风险。

## 主题

- [ ] 网络分段、ACL、防火墙、VPN、NAC 与零信任访问。
- [ ] DNS、DHCP、NTP、邮件、文件共享、远程管理等基础服务安全。
- [ ] 网络设备、负载均衡、代理、无线与堡垒机安全配置。
- [ ] 常见中间件和应用服务的暴露面、认证和日志。
- [ ] 流量分析：协议识别、异常行为、加密流量元数据与取证基础。
- [ ] 配置审计：弱加密、默认账户、管理面暴露、补丁与生命周期。

## 达标产物

- [ ] 网络服务安全基线清单。
- [ ] 管理面暴露风险检查表。
- [ ] 网络分段与访问控制评审模板。
