# 10｜二进制、逆向与漏洞利用基础

目标：在隔离靶场中理解程序执行、保护机制与漏洞根因，支持更准确的风险研判。

## 主题

- [ ] C/C++、汇编、调用约定、内存布局和 ELF/PE 文件格式。
- [ ] 调试与逆向：静态分析、动态分析、符号、调用图和数据流。
- [ ] 常见内存安全问题：越界、释放后使用、整数问题、格式化与逻辑缺陷。
- [ ] 缓解机制：ASLR、DEP/NX、CFG、Canary、沙箱、代码签名与加固。
- [ ] 模糊测试：语料、覆盖率、崩溃去重、最小化和根因定位。
- [ ] 漏洞研究卫生：隔离、样本管理、负责任披露、复现环境记录。

## 达标产物

- [ ] 二进制分析学习路线与靶场清单。
- [ ] 崩溃归因、去重和修复建议模板。
- [ ] 编译器/运行时缓解措施速查表。
