# 08｜云、容器与 DevSecOps

目标：掌握云原生环境的身份、配置、工作负载和软件供应链风险。

## 主题

- [ ] 云责任共担、组织/账号治理、资源层级和资产发现。
- [ ] IAM：身份、角色、策略、临时凭据、联合登录和最小权限。
- [ ] 网络与数据：VPC/VNet、安全组、私网访问、对象存储、密钥和备份。
- [ ] 容器：镜像、运行时、注册表、秘密管理、隔离与网络策略。
- [ ] Kubernetes：认证授权、RBAC、工作负载、准入、网络、审计与多租户。
- [ ] CI/CD 与供应链：代码、依赖、构建、制品、签名、发布和机密管理。
- [ ] IaC：模板审查、策略即代码、漂移检测和变更审计。

## 达标产物

- [ ] 云 IAM 与公开暴露检查清单。
- [ ] 容器/Kubernetes 配置审计清单。
- [ ] CI/CD 供应链威胁建模模板。
