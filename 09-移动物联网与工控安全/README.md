# 09｜移动、IoT 与 OT 安全

目标：理解资源受限、物理可达和高可用环境的特殊信任边界与测试约束。

## 主题

- [ ] Android/iOS：应用沙箱、权限、存储、网络、签名与发布链路。
- [ ] 移动 API、深链、WebView、推送、设备绑定与反欺诈边界。
- [ ] IoT：固件、启动链、调试接口、设备身份、通信协议与 OTA。
- [ ] OT/ICS：资产可见性、分区分域、协议、可用性优先和变更管控。
- [ ] 车联网与边缘设备：远程服务、诊断接口、供应链与物理安全。
- [ ] 测试安全：仿真优先、非生产验证、现场变更窗口和应急回退。

## 达标产物

- [ ] 移动应用安全评估清单。
- [ ] IoT 固件与设备接口审查清单。
- [ ] OT/ICS 测试前置条件与禁止操作清单。
